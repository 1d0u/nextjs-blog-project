## Getting Started

1. Clone the repository
```bash
git clone https://github.com/1d0u/nextjs-blog-project.git
```

2. Install dependencies
```bash
npm install
```

3. Set up environment variables
```bash
cp .env.example .env
```

4. Run the development server
```bash
npm run dev
```
